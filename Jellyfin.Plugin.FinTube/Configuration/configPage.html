YouTube Authentication

Select the browser to read YouTube cookies from for age-restricted content. The browser must be installed on the Jellyfin server and you must be logged into YouTube in that browser.

Music Organizer

When enabled, audio downloads will query MusicBrainz to identify the artist and album. You can review and edit the match before downloading. Files will be organized into Artist/Album/ subfolders automatically.

Post-Download

When enabled and a MusicBrainz match is selected, the YouTube thumbnail will be replaced with the actual album cover from Cover Art Archive, and metadata tags will be rewritten with TagLib# for maximum compatibility.
When enabled, Jellyfin will automatically scan the library for new files after each download completes. Disable if you have a large library and prefer to scan manually.
